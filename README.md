# wikidataProject

Developed as the final project for WGSS 1125. Visit the site [here](https://jebc314.github.io/wikidataProject/).

### Development

The source code is in the folder appDev, so development should happen in that directory.

To run the project for development:

```
npm run dev
```

To build the project for deployment:

```
npm run build
```

### Notes

Due to known interactions between datalist with browser extensions, using the search options with extensions enabled may crash the browser.
